## Ink Anything 是什么? <span id='intro'></span>

Ink Anything 是一款针对 PowerPoint 放映优化的轻量级数字画板，支持墨迹书写、文本输入、图形识别、多点触控手势等功能。

### 三种模式

* **幻灯片模式**：在 PowerPoint 放映时自动打开，支持画笔、橡皮擦、图形工具、文本输入、翻页按键、换页自动换笔迹；墨迹和文本自动保存到每张幻灯片
* **画板模式（黑/白板）**：自由画板，支持添加/切换页面（最多 99 页），每页独立保存墨迹和文本；默认黑板，可在设置中改为白板
* **屏幕画笔模式**：在保留原屏幕内容的同时将鼠标切换为画笔，用于在任意界面上书写标注


## 用法 <span id='usage'></span>

### 画笔设置
* **切换颜色**：点击主界面颜色球，或使用快捷键 Alt+1~6 快速切换
* **调整粗细**：设置 → 画板 → 画笔粗细中调整，默认 5
* **笔锋效果**：设置中可选择根据"速度"或"尾迹"模拟笔锋

### 橡皮擦使用
橡皮擦图标有三种状态，逐次点击循环切换：
1. **灰色（笔画擦除）**：单击一次进入，按笔迹整体擦除
2. **蓝色（范围擦除）**：再单一次，按提示图案范围擦除
3. **恢复画笔**：再单击一次退出橡皮擦
* 手掌按压屏幕可直接擦除（部分屏幕支持）
* 橡皮擦支持擦除文本元素

### 文本输入
* **进入文本模式**：点击浮动工具栏文本图标，或按 Alt+T
* **添加文字**：在画布任意位置点击左键，出现文本框后直接输入
* **提交文本**：按 Enter 提交，或点击其他区域自动提交
* **换行**：按 Shift+Enter 插入换行
* **编辑文本**：在文本模式下双击已提交的文字重新编辑，或右键点击
* **移动文本**：文本模式下左键拖动已提交文字即可移动
* **多选文本**：按住 Ctrl 逐个点击文本元素
* **缩放文本**：点击文字后四角出现缩放手柄，拖动手柄调整大小（范围 8~200）
* **删除文本**：选中后按 Delete 或 Backspace，或用橡皮擦擦除
* **文本颜色**：跟随当前画笔颜色
* **文本大小**：在快捷设置面板中调整（范围 12~72，默认 24）
* **文本光标样式**：在设置中切换箭头或 I 形光标
* **文本选择模式**：在选择模式下按 Alt+T 切换到文本选择范围（仅操作文本，文本框图标浅蓝高亮），再次按 Alt+T 切回墨迹选择范围
* **退出文本模式**：再次点击文本图标、按 Alt+T、或按 Escape

### 绘制预设图形
* **正圆**：在"原点"位置点击确定圆心，拖动鼠标完成绘制
* **椭圆**：在"原点"位置点击拖动，确定长轴和短轴（可选显示焦点）
* **双曲线**：在"原点"位置点击拖动，先确定渐近线，再确定实轴和虚轴（可选显示焦点）
* **抛物线**：在"原点"位置点击，拖动鼠标
* **长方体**：先确定正视图，再次点击确定侧视宽
* **圆锥、圆柱**：一气呵成
* **平面直角坐标系**：有多种预设，图标中两线交点上下左右的长边为实际绘制边
* **空间直角坐标系**：三轴同时绘制
* **平行线**：从一端绘制到另一端，适合绘制匀强电场的电场线，针对特殊角度优化

### 选中操作
* **进入选择模式**：点击选择按钮。选择范围根据进入前状态自动决定：
  * 从画笔模式进入 → 墨迹选择范围（仅操作墨迹，不处理文字点击）
  * 从文本模式进入 → 文本选择范围（仅操作文本，不处理墨迹）
  * 选择范围可通过 Alt+T 切换
* **选择按钮二态切换**：点击进入选择模式 → 再次点击退出选择模式（全选请按 Ctrl+A）
* **操作按钮**：选中后浮动工具栏显示克隆、克隆至新页、旋转（45°/90°）、翻转（水平/垂直）、画笔粗细调整（放大/缩小/恢复）、删除
* **选区拖动**：选中后直接拖动移动位置
* **选区复制**：选中后按住 Ctrl 拖动，可复制到新位置
* **取消选中**：在选中区域外部点击
* **全选**：Ctrl+A 全选当前选择范围内的所有墨迹或文本
* **删除**：选中状态下按 Delete 或 Backspace
* 支持双指手势对选中墨迹进行缩放、移动、旋转

### PPT 放映模式
* **翻页**：屏幕中无笔迹时多指并排上下滑动，或使用控件和键盘方向键
* **墨迹保存**：PPT 中的墨迹和文本自动保存，可移动笔迹文件夹到其他电脑使用
* **隐藏幻灯片检测**：自动检测并提示取消隐藏
* **播放位置记忆**：记忆上次播放位置并提示跳转

### 自动墨迹识别（Ink To Shape）
1. 尽可能画得标准和规范
2. 正圆内画椭圆可识别为截面圆、同心椭圆（适用于天体轨迹）
3. 自动识别同心圆、相切圆（内切、外切）
4. 支持识别：圆、椭圆、三角形、矩形、菱形、平行四边形、正方形
5. 注意：仅在 32 位进程下可用，64 位下自动隐藏

### 浮动工具栏
* 可拖动定位到屏幕任意位置
* 可折叠/展开（Alt+V 或点击表情图标）
* 支持锁定位置，防止误触移动
* 快捷设置菜单：画笔粗细、文本大小、墨迹识别开关、手指模式开关、墨迹保存/加载/回放、倒计时器、随机点名
* 墨迹悬浮工具栏（复制/旋转/镜像/删除）仅在墨迹选中时显示

### 设置页面
点击齿轮图标打开设置，采用 Tab 分页布局：
* **行为**：画板隐藏、手指模式、启动行为
* **画板**：画笔粗细、橡皮类型、光标显示
* **手势**：两指手势（缩放/平移/旋转）
* **墨迹识别**：手绘图形识别开关
* **外观**：主题（浅色/深色/跟随系统）
* **PPT**：PPT 集成设置
* **高级**：特殊屏幕、触摸灵敏度、日志开关
* **自动化**：自动杀进程、自动保存截图/墨迹
* **检查更新**：点击顶部"检查更新"按钮手动检查新版本

所有设置自动保存，部分需重启生效。

### 主题切换
* 支持浅色、深色、跟随系统三种主题
* 黑板/白板模式下自动切换画笔颜色配置（可从 `Colors\Light.ini` / `Dark.ini` 自定义）


## 快捷键 <span id='hotkey'></span>

### 全局快捷键

| 快捷键 | 功能 |
|---|---|
| Alt+S | 切换画笔/鼠标模式 |
| Alt+D | 清屏 |
| Alt+E | 橡皮擦循环切换（笔画擦 → 部分擦 → 退出） |
| Alt+C | 截屏 |
| Alt+V | 显示/隐藏浮动工具栏 |
| Alt+L | 画直线 |
| Alt+T | 文本输入模式（选择模式下切换墨迹↔文本选择范围） |
| Alt+Q | 切换选择模式 |
| Alt+1 | 切换到黑色画笔 |
| Alt+2 | 切换到红色画笔 |
| Alt+3 | 切换到绿色画笔 |
| Alt+4 | 切换到蓝色画笔 |
| Alt+5 | 切换到黄色画笔 |
| Alt+6 | 切换到白色画笔 |
| Ctrl+A | 全选墨迹 |
| Ctrl+Z | 撤销（优先撤销文本操作） |
| Ctrl+Y | 恢复（优先恢复文本操作） |
| Delete / Backspace | 删除选中的墨迹或文本 |
| Shift+Esc | 退出/结束放映 |
| Escape | 退出当前模式 |

### PPT 放映翻页快捷键

| 快捷键 | 功能 |
|---|---|
| Down / PageDown / Right / N / Space | 下一页 |
| Up / PageUp / Left / P | 上一页 |
| 鼠标滚轮 | 上/下翻页 |

### 文本编辑快捷键

| 快捷键 | 功能 |
|---|---|
| Enter | 提交当前文本 |
| Shift+Enter | 在文本中插入换行 |
| Escape | 取消当前文本（删除该文本框） |
| Delete / Backspace | 删除选中的文本元素（非编辑状态） |


## 技巧 <span id='skill'></span>
1. 双击"清屏"按钮可在清屏的同时隐藏画板。
2. 擦错了按 Ctrl+Z 撤销，按 Ctrl+Y 恢复。
3. 空间不够写时，双指向上滑动挪动原有墨迹，或双指缩放腾出空间。
4. 找不到橡皮按钮？笔反过来就是橡皮。也可用手指或倾斜笔尖精确擦除小范围，用手背大范围擦除（手背比手掌更顺滑）。
5. 需要在画布上写字？按 Alt+T 进入文本模式，点击画布即可输入。
6. 快速切换画笔颜色：按 Alt+1~6。
7. 浮动工具栏挡住了内容？按 Alt+V 快速隐藏/显示。
8. 选择模式下按 Alt+T 可在墨迹选择和文本选择范围间切换。
9. 在设置页中点击"检查更新"可手动检查新版本，应用会自动静默升级。
10. 启动时提示"快捷键冲突"说明其他软件占用了相同快捷键。
