## Ink Anything 是什么? <span id='intro'></span>
Ink Anything 画板是一款轻量级画板软件，其针对希沃白板设备进行了特别优化，与预装的"希沃白板 5"软件相比，启动速度大幅度提升（提升5-10 倍），系统资源占用更小，使用体验更佳。  

可以双指缩放拖动甚至旋转，并针对Microsoft PowerPoint 进行优化，可以在放映幻灯片的时候打开白板或黑板进行板书，提高课堂效率。拥有智能墨迹识别技术，可识别圆形等图形，并自动转换。  

统一幻灯片放映时和白板/黑板模式下的墨迹书写体验，形成统一操作习惯。

### Ink Anything 模式

* **幻灯片模式**
    *  在幻灯片模式下支持画笔、橡皮擦、图形工具、文本输入、快捷翻页按键、换页自动换笔迹等功能
    *  自动保存幻灯片下的墨迹和文本，下次打开时将自动恢复，并且随时可修改。
    *  如果有隐藏的幻灯片页面，将询问是否取消隐藏
* **画板模式（黑/白板模式）**
    *  在画板模式下有着一整个类似希沃白板一样的画板
    *  支持添加新页面和页面切换（最多 99 页）
    *  每页独立保存墨迹和文本数据
    *  支持多指书写：黑板模式界面左下角人像图标为切换按钮
	*  默认是黑板，可以在设置中调成白板
* **屏幕画笔模式**
    *  在屏幕画笔模式下可以显示原屏幕内容的同时将鼠标调为画笔书写授课笔迹

### Ink Anything 特性
* 笔细的一头可以在屏幕上留下墨迹，而粗的一头可以当作橡皮擦（按笔画擦除），不必手动切换模式
* 如果觉得翻转笔比较麻烦，可以直接用手指或倾斜笔尖精确擦除小范围墨迹
* 手掌手指并拢，直接放在屏幕上，则是大橡皮（按区域擦除），此动作的识别速度和精度比"希沃白板 5"要高，即碰即擦，无需等待
* 启动速度快
* 更换画笔颜色更方便，不必先点击笔，再切换，清屏也更方便
* 自动查杀希沃部分软件
* 单条或多条墨迹选中后缩放、旋转、移动、克隆
* 全屏幕笔迹双指手势缩放（旋转和拖动也是双指手势）
* **橡皮擦三态切换**：笔画擦除 ↔ 部分擦除 ↔ 退出橡皮擦，支持擦除文本元素
* **文本输入**：支持在画布任意位置添加文字，可拖拽移动、缩放大小、更改颜色，支持撤销/重做，支持 Ctrl+点击多选和 Ctrl+拖动复制
* **墨迹选区增强**：选中墨迹后支持拖动移动、Ctrl+拖动复制；选择按钮支持三态切换（进入选择 → 全选 → 退出），选中状态图标颜色区分（深蓝=全选，浅蓝=部分选中）
* **快捷键冲突检测**：启动时自动检测快捷键冲突并通知提示
* 图形绘制（支持长按一直选中）
  直线、虚线、带箭头直线
  多条平行线，带焦点和不带焦点的椭圆、双曲线、抛物线。
  正圆、虚线圆，圆柱、圆锥、长方体
  坐标系（平面直角坐标系，空间直角坐标系）
* 墨迹转图形，目前可实现智能识别圆、三角形、特殊四边形
  自动转换为规范图形。可自动识别同心圆，相切圆，可自动识别球的截面圆
* **模拟笔锋**：支持根据速度或尾迹模拟笔锋效果，让手写笔迹更自然美观

### Ink Anything 功能  
* **倒计时**：美观的 UI，并可以以接近全屏的大小显示
* **抽奖（随机点名）**：可导入名单（建议搭配 Excel 使用），可设置抽取人数
* **保存墨迹**：默认保存至 `文档\Ink Anything Strokes`
* **截图**：任意模式模式下（包括鼠标）下点击相机图标截图并自动保存至 `图片\Ink Anything Screenshots`，可在设置中开启"截图时自动保存墨迹"
* **幻灯片自动保存墨迹和文本**：默认保存至 `文档\Ink Anything Strokes`
* **墨迹回放**：从头自动书写一遍屏中墨迹
* **主题切换**：支持浅色、深色、跟随系统三种主题


## 用法 <span id='usage'></span>
### 画笔设置
* **颜色**：在"画笔模式"下直接点击主界面的颜色球即可，或使用快捷键 Alt+1~6 快速切换
* **粗细**：点击"设置"齿轮，在"画板"——"画笔粗细"中调整，默认5
* **笔锋**：在"设置"齿轮中，有根据"速度"和"尾迹"绘制笔锋，让笔迹更美观

### 橡皮擦使用
* **笔迹擦除**：单击一次使图标变为灰色可按笔迹擦除内容
* **范围擦除**：单击多一次使图标变蓝色可按提示图案范围擦除
* **退出橡皮擦**：再单击一次恢复画笔模式
* **手掌擦除**：在画笔模式下按压手掌即可起到橡皮擦擦除功能（部分屏幕不支持）
* 橡皮擦也支持擦除文本元素

### 文本输入
* **进入文本模式**：点击浮动工具栏中的文本图标，或按快捷键 Alt+T
* **添加文本**：在画布任意位置鼠标左键点击，出现文本框后直接输入
* **提交文本**：按 Enter 键提交，或点击其他区域自动提交
* **换行**：编辑时按 Shift+Enter 可插入换行
* **编辑文本**：在文本模式下双击已提交的文字，即可重新编辑（也可右键点击）
* **移动文本**：在文本模式下左键拖动已提交的文字，可移动位置
* **多选文本**：按住 Ctrl 逐个点击文本元素，可选中多个文本
* **拖动复制文本**：选中多个文本后，按住 Ctrl 拖动可复制所有选中的文字到新位置
* **缩放文本**：点击文本后四角出现缩放手柄，拖动手柄可调整大小（字体大小范围 8~200）
* **删除文本**：选中状态下按 Delete 或 Backspace 键删除，也可用橡皮擦擦除
* **文本颜色**：跟随当前画笔颜色
* **文本大小**：可在快捷设置面板中调整（范围 12~72，默认 24），也可通过缩放手柄微调
* **文本光标**：可在设置中切换文本模式下的光标样式（箭头或 I 形文本光标）
* **退出文本模式**：再次点击文本图标、按 Alt+T、或按 Escape

### 绘制预设图形
* **正圆**：确定圆心时可选用预设在"原点"位置点击，再拖动鼠标完成正圆绘制
* **抛物线**：可以在"原点"位置点击，再拖动鼠标
* **双曲线**：在"原点"位置点击拖动鼠标，先确定渐近线，然后确定实轴和虚轴
* **椭圆**：在"原点"位置点击拖动鼠标，然后确定长轴和短轴（椭圆和双曲线都可以选择有无显示焦点的预设）
* **长方体**：先确定正视图，然后再次点击，确定侧视宽
* **圆锥、圆柱**：一气呵成
* **平面直角坐标系**：有多种预设，图标中两线交点上下左右的长边为实际绘制边。
* **空间直角坐标系**：三轴同时绘制
* **平行线**：从一端到另一端，适合绘制匀强电场的电场线，并针对特殊角度优化

### 选中操作
* 点击选择按钮进入选中模式，可框选多条墨迹
* **选择按钮三态切换**：首次点击进入选择模式 → 再次点击全选所有墨迹 → 再次点击退出选择模式
* 选中后浮动工具栏显示操作按钮：克隆、克隆至新页、旋转（45°/90°）、翻转（水平/垂直）、画笔粗细调整（放大/缩小/恢复）、删除
* **选区拖动**：选中墨迹后可直接拖动移动位置
* **选区复制**：选中墨迹后按住 Ctrl 拖动，可复制选中墨迹到新位置
* **取消选中**：在选中区域外部点击，自动取消选中
* 支持 Ctrl+A 全选所有墨迹
* 选中状态下按 Delete 或 Backspace 可删除选中墨迹
* 支持双指手势操作选中墨迹（缩放、移动、旋转）
* 选中状态图标颜色区分：深蓝 = 全选，浅蓝 = 部分选中

### PPT 放映模式
* **翻页**：屏幕中无笔迹时可以多指并排上下滑动翻页，以及控件和键盘方向键翻页
* **保存**：ppt中的墨迹和文本自动保存，还可以移动笔迹文件夹到其他电脑正常使用原笔迹
* PPT 模式下打开的内置画板和正常打开的画板一样的，可能需要用户自行关注保存问题
* 支持检测隐藏幻灯片并提示取消隐藏
* 支持记忆上次播放位置并提示跳转

### 自动墨迹识别（Ink To Shape)
1. 尽可能画得标准和规范一些
2. 正圆内画椭圆，可完成截面圆、同心椭圆（适用于天体轨迹）的识别
3. 可自动完成同心圆、相切圆的识别（内切、外切）
4. 可识别圆、椭圆、三角形、矩形、菱形、平行四边形、正方形
5. 注意：仅在 32 位进程下可用，64 位下自动隐藏

### 浮动工具栏
* 可拖动定位到屏幕任意位置
* 可折叠/展开（Alt+V 或点击表情图标）
* 支持锁定位置，防止误触移动
* 快捷设置弹出菜单：画笔粗细、文本大小、墨迹识别开关、手指模式开关、墨迹保存/加载/回放、倒计时器、随机点名

### 主题切换
* 支持浅色、深色、跟随系统三种主题
* 黑板/白板模式下自动切换画笔颜色配置（可从 `Colors\Light.ini` / `Dark.ini` 自定义颜色方案）


## 快捷键 <span id='hotkey'></span>

### 全局快捷键

| 快捷键 | 功能 |
|---|---|
| Alt+S | 切换画笔/鼠标模式 |
| Alt+D | 清屏 |
| Alt+E | 橡皮擦循环切换（笔画擦 → 部分擦 → 退出） |
| Alt+C | 截屏 |
| Alt+V | 显示/隐藏浮动工具栏 |
| Alt+L | 画直线 |
| Alt+T | 文本输入模式 |
| Alt+Q | 切换选择模式 |
| Alt+1 | 切换到黑色画笔 |
| Alt+2 | 切换到红色画笔 |
| Alt+3 | 切换到绿色画笔 |
| Alt+4 | 切换到蓝色画笔 |
| Alt+5 | 切换到黄色画笔 |
| Alt+6 | 切换到白色画笔 |
| Ctrl+A | 全选墨迹 |
| Ctrl+Z | 撤销（优先撤销文本操作） |
| Ctrl+Y | 恢复（优先恢复文本操作） |
| Delete / Backspace | 删除选中的墨迹或文本 |
| Shift+Esc | 退出/结束放映 |
| Escape | 退出当前模式 |

### PPT 放映翻页快捷键

| 快捷键 | 功能 |
|---|---|
| Down / PageDown / Right / N / Space | 下一页 |
| Up / PageUp / Left / P | 上一页 |
| 鼠标滚轮 | 上/下翻页 |

### 文本编辑快捷键

| 快捷键 | 功能 |
|---|---|
| Enter | 提交当前文本 |
| Shift+Enter | 在文本中插入换行 |
| Escape | 取消当前文本（删除该文本框） |
| Delete / Backspace | 删除选中的文本元素（非编辑状态） |


## 技巧 <span id='skill'></span>
1. 双击"清屏"按钮即可在清屏的同时隐藏画板，相当于点击"清屏"和"隐藏画板"两个按钮。
2. 一不小心擦错了，点击"撤销"按钮或按 Ctrl+Z 即可撤销，再次点击或按 Ctrl+Y 即可恢复。
3. 如果需要关闭Ink Anything画板，可在设置中找到关闭按钮。
4. 在任意时刻，点击"黑板"按钮即可进入黑板。
5. 空间不够写时，可以双指向上滑动来挪动原有墨迹，也可以双指缩放以腾出空间。
6. 找不到"橡皮"按钮？笔反过来就是橡皮了！如果觉得翻转笔比较麻烦，可以直接用手指或倾斜笔尖精确擦除小范围墨迹。大范围擦除可以通过并拢手指使用手掌或手背，甚至黑板擦、抹布来完成。建议使用手背而非手掌，因为手掌上汗较多，在屏幕上滑动时摩擦较大，且较大的接触面积可能导致误识别为多指（多指缩放）的问题。
7. 需要在画布上写字？按 Alt+T 进入文本模式，点击画布任意位置即可输入文字，支持拖拽移动和缩放大小。
8. 快速切换画笔颜色：按 Alt+1~6 可在黑、红、绿、蓝、黄、白六种颜色间快速切换。
9. 浮动工具栏挡住了内容？按 Alt+V 可以快速隐藏/显示浮动工具栏。
10. 需要复制多个文字元素？按住 Ctrl 逐个点击选中多个文字，再按住 Ctrl 拖动即可复制全部。
11. 需要快速选中所有墨迹？按 Ctrl+A 一键全选，或连按两次选择按钮。
12. 启动时如果提示"快捷键冲突"，说明其他软件占用了相同快捷键，可在该软件中修改或忽略。
