# Ink-Anything

基于 WPF/C# 的轻量级数字画板，支持文本输入，针对希沃白板和 PowerPoint 放映深度优化。

</div>

[English](README_EN.md) | 中文

## 特性

- **Microsoft PowerPoint 深度集成**：放映时自动切换画板模式，支持翻页控制、墨迹/文本保存、隐藏幻灯片检测、播放位置记忆
- **三种模式**：幻灯片模式（自动保存墨迹和文本）、画板模式（黑/白板，最多 99 页）、屏幕画笔模式
- **主动笔/压感支持**，笔尖书写笔尾擦除（希沃白板原生不支持此功能）
- **多点触控手势**：双指缩放/平移/旋转，手指触摸擦除
- **白板多页管理**，每页独立保存墨迹和文本数据
- **文本输入**：画布任意位置添加文字，支持拖拽移动、缩放大小、更改颜色、撤销/重做、Ctrl+多选
- **手绘图形识别**：自动识别圆、三角形、矩形等图形并转换为规范图形；支持同心圆、相切圆、截面圆识别
- **图形绘制工具**：直线/箭头/虚线、椭圆/双曲线/抛物线（可选焦点）、正圆/虚线圆、圆柱/圆锥/长方体、平面/空间直角坐标系、平行线
- **选择模式分离**：进入选择模式时自动区分墨迹选择与文本选择范围，Alt+T 切换选择范围
- **橡皮擦三态切换**：笔画擦除 ↔ 部分擦除 ↔ 退出橡皮擦，支持擦除文本元素
- **墨迹选区增强**：选中后拖动移动、Ctrl+拖动复制，选择按钮二态切换（进入选择 ↔ 退出），Ctrl+A 全选
- **模拟笔锋**：根据速度或尾迹模拟笔锋效果
- **设置页面 Tab 分页布局**：分类标签页管理所有设置项，内置检查更新按钮
- **自动更新**：从 GitHub Releases 获取更新，使用 Inno Setup 安装包静默升级
- **浮动工具栏**：可拖拽定位、折叠/展开、锁定位置，内置快捷设置菜单
- **教学辅助工具**：倒计时器、随机点名（可导入名单）
- **墨迹保存/加载/回放**，截图自动保存
- **深色/浅色/跟随系统主题切换**
- **全局热键**：Alt+S/D/E/C/V/L/T/Q，Alt+1~6 快速切换画笔颜色，Ctrl+A/Z/Y
- **快捷键冲突检测**：启动时自动检测并提示
- **开机自启动**

## 键盘快捷键

| 快捷键 | 功能 |
|---|---|
| Alt+S | 切换画笔/鼠标模式 |
| Alt+D | 清屏 |
| Alt+E | 橡皮擦循环切换（笔画擦 → 部分擦 → 退出） |
| Alt+C | 截屏 |
| Alt+V | 显示/隐藏浮动工具栏 |
| Alt+L | 画直线 |
| Alt+T | 文本输入模式（选择模式下切换墨迹↔文本选择范围） |
| Alt+Q | 切换选择模式 |
| Alt+1~6 | 切换画笔颜色（黑/红/绿/蓝/黄/白） |
| Ctrl+A | 全选墨迹 |
| Ctrl+Z | 撤销 |
| Ctrl+Y | 恢复 |
| Shift+Esc | 退出/结束放映 |
| Escape | 退出当前模式 |

## 运行环境

- Windows 10 及以上
- .NET Framework 4.7.2
- Microsoft Office（PPT 集成需要）
