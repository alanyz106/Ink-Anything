# Ink-Anything

A fantastic Ink Anything in WPF/C#, with fantastic support for Seewo Boards.

</div>

## 特性

- 对 Microsoft PowerPoint 有优化支持，放映时自动切换画板模式，支持翻页控制和墨迹保存
- 支持 Active Pen（压感支持）
- 笔细的一头写字，反过来粗的一头是橡皮擦（希沃白板本身不支持此功能）
- 手指直接触摸擦除墨迹
- 多点触控手势支持（缩放、平移、旋转）
- 白板多页管理，支持无限画布
- 手绘图形识别（圆、矩形、三角形等），自动转换为规范图形
- **文本输入**：支持在画布任意位置添加文字，可拖拽移动、缩放大小、更改颜色，支持撤销/重做，白板翻页和 PPT 放映中自动保存
- **橡皮擦三态切换**：笔画擦除 ↔ 部分擦除 ↔ 退出橡皮擦，支持擦除文本元素
- 浮动工具栏，支持锁定/解锁位置
- 倒计时器、随机点名等教学辅助工具
- 截图并自动保存
- 墨迹保存与加载
- 墨迹回放功能
- 深色/浅色主题切换
- 开机自启动
- 全局热键支持
- 对其他红外触控屏也可提供相似功能

## 运行环境

- Windows 10 及以上
- .NET Framework 4.7.2
- Microsoft Office（PPT 集成需要）
